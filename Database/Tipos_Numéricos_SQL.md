# Tipos Numéricos do SQL:

- Numérico:

  - Sub-Divisão: Inteiro, Real, Lógico

    - Tipos:

    Inteiro: TinyInt, SmallInt, Int, MediumInt, BigInt;

    Real: Decimal, Float, Double, Real;

    Lógico: Bit, Boolean;

- Data/ Tempo:

  - Tipos:

  Date, DateTime, TimeStamp, Time, Year;

- Literal :

  - Sub-Divisão: Caractere, Texto, Binário, Coleção

    - Tipos:

    Caractere: Char, VarChar;

    Texto: TinyText, Text, MediumText, LongText;

    Binário: TinyBlob, Blob, MediumBlob, LongBlob;

    Coleção: Enum, Set;

- Espacial:

  - Tipos:

  Geometry, Point, Polygon, MultiPolygon;
